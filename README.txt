💡 Optimiza automáticamente las imágenes de tu sitio para mejorar el rendimiento y velocidad de carga.

== 📋 Descripción ==

OptiImagen analiza y optimiza automáticamente las imágenes que subís a tu sitio WordPress, reduciendo su peso sin perder calidad.  
Aumentá la velocidad de tu sitio y mejorá el SEO sin esfuerzo manual.

== ⚙️ Instalación ==

1. Subí la carpeta del plugin a /wp-content/plugins/optiimagen/
2. Activá el plugin desde el panel de administración de WordPress
3. ¡Listo! Las nuevas imágenes se optimizarán automáticamente 🧠

== ❓ Preguntas Frecuentes ==

= ¿Se optimizan automáticamente todas las imágenes nuevas? =
Sí, al subirlas se procesan sin que tengas que hacer nada.

= ¿Puedo optimizar imágenes anteriores? =
En esta versión solo se optimizan las nuevas, pero estamos trabajando en una opción para el historial.
